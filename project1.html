1. Importing Data

Data came from two CSV files:

  • Orders Table: Customer & order details.
  • Order Details Table: Payment modes, amounts, products.

2. Data Cleaning and Transformation

Using Power Query to:

  • Change data types (text to date, etc.)
  • Remove unnecessary columns & duplicates.
  • Split columns and calculate Average Order Value.

3. Building Relationships

Linked "Order ID" between tables, set One-to-Many relationships in Power BI.

4. Creating the Dashboard

Added charts (Profit by Month, Profit by Sub-Category), conditional formatting, cards, and slicers.

5. Customizing the Dashboard

Used background images, format painter, consistent fonts, and cohesive layout.

6. Profit by Sub-Category

Stacked Bar Chart shows sub-category profits (Top N filter applied).

7. Quantity by Category

Pie chart for Quantity by Category, and Payment Mode.

8. Key Metrics

Cards for Total Amount, Quantity, Profit, and Average Order Value.

9. Additional Charts

Top N Statuses and Top Customers visualized.

10. Adding Slicers

Slicers for Quarters and States to filter the dashboard interactively.

11. Final Adjustments and Testing

Ensured proper alignment, styling, interactivity.

12. Final Outcome

Comprehensive dashboard with cards, slicers, and multiple visualizations.

13. Features of the Project

  • Comprehensive data visualization.
  • Dynamic filters.
  • Data modeling & relationships.
  • Custom metrics (DAX).
  • Polished design.
  • Advanced data transformation.
